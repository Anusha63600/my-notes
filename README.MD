📝 Notes App

A simple and interactive Notes App built using HTML, CSS, and JavaScript.
This app allows users to create, edit, delete, and manage notes with useful features like dark mode, search, and important notes.

---

🚀 Features

- ➕ Add new notes
- ✏️ Edit existing notes
- ❌ Delete notes
- ⭐ Mark notes as important
- 📌 Important notes pinned on top
- 🔍 Search notes (case-insensitive)
- 🌙 Dark mode / Light mode toggle
- 📅 Date & time for each note
- 💾 Data stored in browser using localStorage

---

🛠️ Technologies Used

- HTML
- CSS
- JavaScript
